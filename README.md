# Simple Calculator

A simple calculator built with HTML, CSS, and JavaScript. It supports the four basic arithmetic operations and was created to practice DOM manipulation, application state management, and JavaScript fundamentals.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Chained calculations
- Division-by-zero handling
- Clear button
- Minimalist interface

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)

## Installation

1. Clone the repository.
2. Open `index.html` in your web browser.

## Usage

Enter numbers using the calculator buttons, select an operator, and press `=` to evaluate the expression. Press `CLEAR` to reset the calculator.
